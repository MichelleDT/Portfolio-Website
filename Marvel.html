Marvel Quiz

  • Themed Game Interface
  • Dynamic Javascript Content

Live Demo Github Repo

Description:

Project Brief: I was tasked with creating a timed quiz with a specific theme. Being an avid gamer and comic book nerd, I wanted to create a quiz based on my favourite Marvel superheroes ‘The Avengers’.


Problem: The example quiz layout indicated the outcome with words only, and never indicated what the correct answer was, robbing the player of learning something.


Approach: LWith this in mind I knew I wanted a themed and streamlined user interface, with the elements adding to the overall arching superhero theme and streamlined game aesthetic. Being an avid quiz taker I wanted to implement the colour change indicator (green for correct, red for incorrect) element into my quiz.


Solution: My game in the end not only indicated the outcome with the color changing element, but also displayed the correct answer to the user.
