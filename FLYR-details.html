FLYR

Live Demo Github Repo

Description:

Project Brief: We were tasked to work in a group and develop a user interface that utilizes API’s to solve a users problem in a new way. Our team loved travelling and wanted to create an interface that assisted the user in the biggest pain point, the research and planning of a trip.


Problem: From researching flight, accomodation, activities to do in the country, weather, and finding out what was happening in the news, this all required research and often meant visiting multiple information sources to fulfill their information needs. Additionally once you find this information there was no way to view all this information in one interface in order to get a holistic view of your trip.


Approach: Working in a group and tackling such a big task, we all played to our strengths in order to maximise efficiency and ensure the best end product. One team member focused on the back end, integrating the API and database, whilst I used my design background to create the UI and UX experience of the interface and assisted the third team member working on the front end.


Solution: Within 2 weeks we were able to create an MVP version of the FLYR interface, which pulled data from 5 API’s to display basic flight information, top activities, weather and news relating to the user’s input end destination. This MVP is just the start and we will continue to work on and develop FLYR to become a better interface.
