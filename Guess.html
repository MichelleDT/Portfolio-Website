Live Demo Github Repo

Description:

Project Brief: Planet X is a Word Guessing Game modeled of the game metrics of Hangman. The game mechanics are simple, you have 10 chances to try and spell the space themed mystery word. Doing so before those 10 guesses runs out results in a win and failing to do so in a loss.


Problem: Seeing as the game concept of Hangman has been done so many times before I wanted to create something interesting and different for the user, something that would set my game apart from others.


Approach: With this in mind I knew I wanted a themed and streamlined user interface, where players click the letter board rather than type their response, and all elements added to the overall arching space theme and streamlined game aesthetic. Being an avid gamer and familiar with the game modes system, I wanted to implement this element to the game.


Solution: As a result I added in a Hard Mode where players now only have 6 chances to guess the correct mystery word. Additionally, gaming is often a collaborative team experience, thus I wanted to create a 2 player mode where users could play with a loved one.
